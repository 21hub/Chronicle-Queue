Chronicle

<pre>

    &lt;groupId>net.openhft&lt;/groupId>
    &lt;artifactId>chronicle&lt;/artifactId>
    &lt;version>2.0.1&lt;/version>
&lt;/dependency>
</pre>

For working examples, have a look at the unit tests and demo module.
